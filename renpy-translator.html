Переводчик для игр на Ren'Py

NovaTextVN — это переводчик для игр на Ren'Py, который помогает переводить визуальные новеллы на русский язык без ручной сборки. Пользователь получает готовый архив, устанавливает его в папку игры и дальше уже может играть с real-time переводом, pre-scan и кэшем строк.

Почему это удобно

  • не нужно собирать проект вручную
  • есть готовый релизный zip
  • основной сценарий установки понятен обычному игроку

Что умеет NovaTextVN

  • переводить текст прямо во время игры
  • ускорять повторные строки через кэш
  • подготавливать длинные игры через pre-scan

Для кого подходит

  • для игроков visual novel
  • для тех, кто ищет перевод игр на Ren'Py на русский
  • для тех, кому нужен готовый переводчик без лишней настройки

Быстрый маршрут

Если тебе нужен быстрый и понятный путь, маршрут такой: открыть страницу скачивания, установить архив по инструкции, проверить игру в real-time режиме и затем при необходимости использовать pre-scan для длинного прохождения.
