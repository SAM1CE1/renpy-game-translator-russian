How to install NovaTextVN

Below is the practical install flow for the public release, without manual builds and without editing game source files.

Installation steps

01

Download novatextvn_dropin.zip

Open GitHub Releases and download the latest public NovaTextVN archive.

If GitHub shows multiple links, use the attached ZIP asset from the release itself.

02

Extract the archive

Open the ZIP and extract its contents into any temporary folder on disk.

After extraction, you should see novatextvn_patch.rpa and novatextvn_core.exe.

03

Open the game folder

Find the installed Ren'Py game where you want to enable translation.

If needed, open the folder through the game shortcut or launcher.

04

Go into game/

Inside the game folder, open game/ — this is where NovaTextVN's public release belongs.

Do not place the files into the game root or into a nested game/game/ directory.

05

Copy the release files

Move novatextvn_patch.rpa and novatextvn_core.exe into game/.

If an older NovaTextVN version is already present, review the existing files and update carefully.

06

Start the game and press F10

After the game starts, open the NovaTextVN menu with F10 and enable translation.

If the menu does not appear, double-check the install path and the compatibility notes.

Common install mistakes

Files were copied into the wrong folder

The most common mistake is placing the files into the game root instead of game/.

The archive was not fully extracted

Make sure the two release files are inside game/, not the ZIP archive itself.

An older version was mixed with the new one

If you are updating, make sure there are no leftover duplicate files from an older build.

The game uses a non-standard structure

Some Ren'Py projects heavily customize their structure or UI. In those cases, review compatibility first.

How to remove NovaTextVN

If you want to return the game to its original state, remove the two public-release files from game/.

game/
  novatextvn_patch.rpa
  novatextvn_core.exe
