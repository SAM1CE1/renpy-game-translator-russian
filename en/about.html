About
Project overview

What NovaTextVN is

NovaTextVN helps you read Ren'Py games in Russian without a manual build. The public Base release is installed from a ready archive: copy the files into game/, launch the game, and enable translation with F10.

Download BaseOpen install guide

What it is

A finished public release for Ren'Py games, not a source-code project you have to build yourself.

What it fits

Mainly visual novels and other Ren'Py projects where story scenes, dialogue, and reading flow matter most.

What gets installed

Two files go into game/: novatextvn_patch.rpa and novatextvn_core.exe.

What Base gives you

A quick start, a simple install path, F10 activation, and an easy way to check your game.

How it works in practice

01

Download the archive

Get the public release from GitHub Releases.

02

Copy the files into game/

The release goes into the game folder without a manual build process.

03

Launch the game and press F10

That is where you enable translation and verify the workflow.

What Base already covers

  • A simple path from download to first launch.
  • In-game translation without a complicated external workflow.
  • Activation and verification directly through F10.
  • A quick way to see whether NovaTextVN fits your game.

What to keep in mind

  • Compatibility still depends on the specific game and its interface.
  • Heavily customized projects are safer to test on a backup copy first.
  • If you need a more controlled result, PRO has its own dedicated page.
