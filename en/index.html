What it is

A practical translator for Ren'Py

NovaTextVN is a practical way to add Russian translation to a Ren'Py game without manually building the product. The public version is distributed as a ready ZIP archive.

Built for visual novels and story-heavy games

The site is aimed at players who want to read Ren'Py games in Russian without a long setup process. The Base version is designed for the first full playthrough and story comprehension.

It works best in games where dialogue, story choices, and the standard Ren'Py interface are the main focus.

What gets added to the game

The public archive is installed into game/ and adds two working files: novatextvn_patch.rpa and novatextvn_core.exe.

This is an honest public release model: you receive the finished product for installation, not public source code.

How to install NovaTextVN into a Ren'Py game

The install flow is short and does not require a source build, code editing, or a separate launcher.

01

Download the latest release

Open GitHub Releases and download the novatextvn_dropin.zip archive.

02

Open the game folder

Find the installed Ren'Py game where you want translation enabled.

03

Go into game/

This is the folder where NovaTextVN's working files belong.

04

Copy the release files

Place novatextvn_patch.rpa and novatextvn_core.exe into game/.

05

Start the game normally

No custom launcher is required. After copying the files, run the game as usual.

06

Press F10 and enable translation

Open the NovaTextVN menu, check status, and turn translation on.

Important

If the menu does not appear, first confirm that the files are inside game/, not in the game root or in an extra nested folder created during extraction.
