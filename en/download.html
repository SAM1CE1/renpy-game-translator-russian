Is this safe to install?

The public Base release does not require a manual build or a separate launcher. Installation into a game consists of copying only two files from the official archive.

  • Official download point: GitHub Releases of NovaTextVN-Releases.
  • Working release files: novatextvn_patch.rpa and novatextvn_core.exe.
  • Both files belong inside the specific Ren'Py game's game/ folder.
  • To roll back, remove those two files from game/.
game/
novatextvn_patch.rpa
novatextvn_core.exe

Fast rollback

If you want to restore the game to its original state, remove both public-release files from game/ and launch the game normally again.
