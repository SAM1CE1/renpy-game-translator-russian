Безопасно ли это ставить?

Публичный Base-релиз не требует ручной сборки и не просит отдельный лаунчер. Для установки в игру копируются только два файла из официального архива.

  • Официальная точка загрузки: GitHub Releases репозитория NovaTextVN-Releases.
  • Рабочие файлы релиза: novatextvn_patch.rpa и novatextvn_core.exe.
  • Оба файла должны лежать именно в папке game/ конкретной Ren'Py-игры.
  • Чтобы откатить установку, достаточно удалить эти два файла из game/.
game/
novatextvn_patch.rpa
novatextvn_core.exe

Быстрый откат

Если хотите вернуть игру в исходное состояние, удалите из game/ оба файла публичного релиза и снова запустите игру обычным способом.
